# Menu Admin

관리자 메뉴 관리 API.

## URL

| Method | URL               | 권한          | 설명    |
| ------ | ----------------- | ----------- | ----- |
| GET    | /admin/menus      | MENU_READ   | 메뉴 조회 |
| POST   | /admin/menus      | MENU_CREATE | 메뉴 생성 |
| PATCH  | /admin/menus/{id} | MENU_UPDATE | 메뉴 수정 |
| DELETE | /admin/menus/{id} | MENU_DELETE | 메뉴 삭제 |

## 구조

```text
admin/menu
├── MenuAdminController
├── dto
│   ├── MenuRequest
│   └── MenuResponse
├── service
│   └── MenuAdminService
├── domain
│   └── Menu
└── repository
    └── MenuRepository
```

## 흐름

Controller
→ Service
→ Repository
→ Menu(Entity)

## 권한

```text
MENU_READ
MENU_CREATE
MENU_UPDATE
MENU_DELETE
```

Spring Security의 `@PreAuthorize`를 사용하여 접근을 제어한다.
