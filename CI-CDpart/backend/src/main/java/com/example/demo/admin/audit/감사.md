# Menu Admin Module 정리

## 1. 전체 구조

```

com.example.demo
└── admin
├── controller (MenuAdminController)
├── service (MenuAdminService)
└── dto
├── MenuRequest
└── MenuResponse

````

---

## 2. 문제 원인 (기존 에러)

### ❌ 1) 패키지 경로 불일치
- DTO 실제 위치: `com.example.demo.admin.dto`
- Controller import: `com.example.demo.admin.menu.dto`

➡ 패키지 구조가 서로 다름

---

### ❌ 2) DTO 중복 정의
- `MenuResponse` 클래스 2개 존재

➡ 컴파일 충돌 발생

---

### ❌ 3) Service 미구현
- `MenuAdminService` 클래스 없음

➡ Controller에서 의존성 주입 실패

---

### ❌ 4) DTO Getter/Setter 없음
- Spring JSON 바인딩 불가

---

## 3. 수정 방향 (정답 구조)

### ✔ DTO 패키지 통일
```java
package com.example.demo.admin.dto;
````

---

### ✔ Controller import 수정

```java
import com.example.demo.admin.dto.MenuRequest;
import com.example.demo.admin.dto.MenuResponse;
import com.example.demo.admin.service.MenuAdminService;
```

---

### ✔ DTO 구조 (Lombok 권장)

#### MenuRequest

```java
@Getter
@Setter
public class MenuRequest {
    private String name;
    private int price;
}
```

#### MenuResponse

```java
@Getter
public class MenuResponse {
    private Long id;
    private String name;
    private int price;
}
```

---

## 4. Service 구조

```java
@Service
public class MenuAdminService {

    public List<MenuResponse> getMenus() {
        return List.of();
    }

    public void createMenu(MenuRequest request) {}

    public void updateMenu(Long id, MenuRequest request) {}

    public void deleteMenu(Long id) {}
}
```

---

## 5. Controller 구조

```java
@RestController
@RequestMapping("/admin/menus")
@RequiredArgsConstructor
public class MenuAdminController {

    private final MenuAdminService menuAdminService;

    @GetMapping
    public List<MenuResponse> getMenus() {
        return menuAdminService.getMenus();
    }

    @PostMapping
    public void createMenu(@RequestBody MenuRequest request) {
        menuAdminService.createMenu(request);
    }

    @PatchMapping("/{id}")
    public void updateMenu(@PathVariable Long id,
                           @RequestBody MenuRequest request) {
        menuAdminService.updateMenu(id, request);
    }

    @DeleteMapping("/{id}")
    public void deleteMenu(@PathVariable Long id) {
        menuAdminService.deleteMenu(id);
    }
}
```

---

## 6. 핵심 정리

* DTO 패키지는 반드시 Controller와 동일 구조 유지
* DTO는 Lombok 또는 getter/setter 필수
* Service는 반드시 존재해야 Controller 컴파일 가능
* 같은 클래스 중복 정의 금지
