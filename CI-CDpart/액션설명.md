# 2️⃣ CI 단계 역할 정의 표

| 단계             | 목적       | 도구                      | 실패 시 영향   |
| -------------- | -------- | ----------------------- | --------- |
| Checkout       | 코드 가져오기  | actions/checkout        | 즉시 실패     |
| Setup          | 실행환경 구성  | setup-node / setup-java | 즉시 실패     |
| Install        | 의존성 설치   | npm / gradle            | build 불가  |
| Lint           | 코드 품질 검사 | ESLint / Checkstyle     | 품질 차단     |
| Test           | 단위 테스트   | JUnit / Jest            | 배포 차단     |
| Build          | 컴파일/번들   | Gradle / Vite           | 이미지 생성 불가 |
| Docker Build   | 컨테이너 생성  | Docker                  | 배포 불가     |
| Security Scan  | 취약점 검사   | Trivy                   | 선택적 차단    |
| Push           | 이미지 업로드  | GHCR / ECR              | 배포 불가     |
| Trigger Deploy | 배포 호출    | Terraform               | CD 단계     |

# ✅ 3. Backend CI (개선된 실무형)

```yaml
name: backend-ci

on:
  push:
    branches: ["main", "develop"]

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Setup JDK
        uses: actions/setup-java@v4
        with:
          distribution: temurin
          java-version: 17

      - name: Grant permission
        run: chmod +x gradlew

      - name: Test & Build
        run: ./gradlew build

      - name: Docker Build
        run: docker build -t myapp-backend ./backend

      - name: Login GHCR
        uses: docker/login-action@v3
        with:
          registry: ghcr.io
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}

      - name: Push Image
        run: |
          docker tag myapp-backend ghcr.io/org/myapp-backend:latest
          docker push ghcr.io/org/myapp-backend:latest
```

# ✅ 4. Frontend CI (개선된 실무형)

```yaml
name: frontend-ci

on:
  push:
    branches: ["main"]

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
      - uses: actions/checkout@v4

      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: 20

      - name: Install
        run: npm ci

      - name: Lint
        run: npm run lint

      - name: Test
        run: npm run test -- --watchAll=false

      - name: Build
        run: npm run build

      - name: Docker Build
        run: docker build -t myapp-frontend ./frontend

      - name: Login GHCR
        uses: docker/login-action@v3
        with:
          registry: ghcr.io
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}

      - name: Push Image
        run: |
          docker tag myapp-frontend ghcr.io/org/myapp-frontend:latest
          docker push ghcr.io/org/myapp-frontend:latest
```
